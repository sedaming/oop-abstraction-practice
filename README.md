# Python OOP Practice Projects

## Overview
This repository contains practice projects focused on Object-Oriented Programming concepts in Python.

## Projects

### 1️⃣ Device Energy Consumption
- Demonstrates abstraction using device classes
- Includes implementations for Phone and Laptop

### 2️⃣ Car Abstraction
- Demonstrates abstraction and property usage
- Includes fuel validation logic
- Supports speed increase and decrease

## Concepts Covered
- Abstraction (ABC)
- Inheritance
- Method Overriding
- Encapsulation
- Property & Setter
